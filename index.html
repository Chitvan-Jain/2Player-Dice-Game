Dice Game Rules

  1. Objective: Reach the winning score first which is 100 points.
  2. Setup: Two players with current and total scores displayed.
  3. Turns:
    1. Players take turns rolling a single dice.
    2. Roll the dice by clicking "Roll Dice".
    3. If the dice shows 2-6, add the number to the current score.
    4. If the dice shows 1, reset the current score to 0 and switch turns.
  4. Hold:
    1. At any time, a player can "Hold" to add the current score to their total score, reset the current score to 0, and switch turns.
  5. Winning: The first player to reach the winning score wins, and a message is displayed.
  6. Switching Players: Highlight the active player and ensure the non-active player is not highlighted.
  7. Restarting: Click "New Game" to reset all scores and start anew.
